Usage Guidelines

Do's

  • ✓ Use liquid glass effects for cards and containers
  • ✓ Apply glitch effect only to main logo and hero titles
  • ✓ Maintain consistent spacing and typography hierarchy
  • ✓ Use gradients for text highlights and backgrounds
  • ✓ Keep animations smooth and purposeful

Don'ts

  • ✗ Don't overuse glitch effects on body text
  • ✗ Don't use colors outside the defined palette
  • ✗ Don't make animations too fast or distracting
  • ✗ Don't use solid backgrounds instead of liquid glass
  • ✗ Don't mix different border radius values
